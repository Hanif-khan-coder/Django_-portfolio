# 💼 Personal Portfolio Website

A simple and responsive personal portfolio website built with Django / HTML / CSS.

---

## 🚀 Features

- 🏠 Home page with introduction
- 👨‍💻 About section
- 📁 Projects section (showcase work)
- 📞 Contact page
- 🔗 Navigation between pages (smooth routing)
- 📩 Contact form submits data to backend
- 📊 Backend admin panel to view submitted messages

---

## ⚙️ Functionality

- Users can click and move between different pages easily
- Contact form data is stored in backend database
- Admin can view all submitted form messages
- Fully dynamic website with backend support

---

## 🛠️ Tech Stack

- Python
- Django
- HTML
- CSS
- SQLite (Database)

## 📂 Project Structure
